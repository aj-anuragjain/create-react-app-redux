# Boilerplate with Create-React-App && Redux

## Development Commands

**Yarn**

```
yarn install
yarn start
```

**npm**

```
npm install
npm start
```

## Get the production build

**Yarn**

```
yarn build
```

**npm**

```
npm build
```

## Run the production server

**Yarn**

```
yarn run start:prod
```

**npm**

```
npm run start:prod
```
